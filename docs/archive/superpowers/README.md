# Archived Superpowers Docs

## Folder Này Chứa Gì

Đây là cây `docs/superpowers/` cũ đã được chuyển vào archive. Nội dung còn lại ưu tiên spec chi tiết, audit UI dạng Markdown và handoff cho agent; binary asset/prototype HTML đã được loại khỏi nhánh docs.

## Khi Nào Nên Đọc

- Khi cần bối cảnh lịch sử sâu hơn root docs.
- Khi cần xem spec gốc ngày 2026-06-09 đến 2026-06-19.
- Khi cần audit UI cũ hoặc context prototype cũ ở dạng Markdown để so sánh.

## Lưu Ý

- Root docs mới là source-of-truth đọc nhanh.
- Một số file cũ dùng từ như mock/MVP/placeholder vì được viết cho nội bộ/agent; không dùng nguyên văn làm copy UI người dùng.
- Nếu có mâu thuẫn, ưu tiên root docs và code hiện tại.
