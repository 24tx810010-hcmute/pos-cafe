# Screenshots

## Folder Này Chứa Gì

Folder này chứa ảnh UI hiện tại của app để audit, so sánh và đưa vào tool/agent redesign.

- `current/`: bộ screenshot mới nhất đã chụp từ app hiện tại.

## Khi Nào Nên Đọc

- Khi cần hiểu UI hiện tại mà không chạy app.
- Khi chuẩn bị prompt cho Gemini/Stitch/Figma/agent redesign.
- Khi so sánh trước/sau UI rework.

## Quy Tắc Duy Trì

- Chỉ giữ một bộ `current/` đại diện trạng thái mới nhất.
- Bộ ảnh cũ nếu cần giữ thì chuyển vào `docs/archive/`.
- Khi UI core thay đổi đáng kể, chụp lại floor, order, payment và takeaway trước.
